### 🚀 Mis Proyectos Destacados

Aquí hay una selección de proyectos en los que he trabajado.

* **[TaskFlow - Gestor de Tareas](https://github.com/hnrylvo/To-Do)**
    * Una PWA Full-Stack para gestionar tareas con React, Node.js y SQLite.
    * Incluye autenticación JWT, panel de analíticas, prioridades y modo oscuro.

* **[InmoMarket - Plataforma Web](https://github.com/hnrylvo/InmoMarket-Frontend)**
    * Plataforma web de anuncios inmobiliarios construida con React, Node.js y MongoDB.
    * Enfocada en una interfaz responsive y paridad de funciones con la versión móvil.

* **[InmoMarket - App Móvil](https://github.com/milianz/ProyectoPDM)**
    * Aplicación nativa de Android para anuncios inmobiliarios usando Kotlin y Jetpack Compose.
    * Permite a los usuarios publicar, buscar y contactar directamente a los vendedores.
